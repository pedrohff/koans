# Koans

Programming exercises to keep my brain up to date with the languages I like to work with and mess with coding challenges, data structures, code complexity, performance, concurrency and such. The idea behind it is to build these scripts by using no IDE, just a simple Nvim linter, so I can keep fresh the basics of my preferred programming languages.
